# Filip Lněnička - Personal Portfolio

This is a modern, interactive personal portfolio website built with React, Vite, and Tailwind CSS.

## Features

- **Responsive Design:** Optimized for all devices.
- **Animations:** Powered by Framer Motion and Anime.js.
- **Gallery:** Photography portfolio with masonry layout.
- **Pages:**
  - Home
  - Web Creation
  - Marketing
  - Photography
  - AI
  - Contact
  - Links (Linktree style)

## Getting Started

1.  **Install Dependencies:**
    ```bash
    npm install
    ```

2.  **Run Development Server:**
    ```bash
    npm run dev
    ```

3.  **Build for Production:**
    ```bash
    npm run build
    ```

## Customization

-   **Images:** Place images in `public/images/`.
-   **Content:** Edit pages in `src/pages/`.
-   **Styles:** Tailwind CSS is configured in `src/index.css`.
